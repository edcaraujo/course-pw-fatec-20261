## Requisitos

Como método de avaliação para a **P2**, os alunos terão que desenvolver uma API atendendo aos seguintes requisitos:

- Deve utilizar alguma biblioteca de ORM
- Deve possuir autenticação
- Deve possuir documentação ([APIdog](https://apidog.com/pt-BR/))
- Deve possuir testes unitários
- Deve ser disponibilizado no Github
- Deve ser desenvolvido em equipe de 2-4 pessoas.

---

## Extra

 - Configurar Dockerfile ([Docker](https://www.docker.com/))
 - Configurar testes e2e ([APIdog](https://apidog.com/pt-BR/))
 - Configurar deploy ([Render](https://render.com/))
